ยกเลิก code ที่ทำก่อนหน้านี้ เพราะ Ui ยังไม่ตรงตามรูปภาพและ Promp ที่เขียนไว้ 
ให้ทำตาม promp ด้านล่างนี้ทุกตัวอักษรอย่างละเอียดเข้มงวดที่สุด

สร้างแอพ MindVerse เขียน code เป็น main.dart เป็นแอพ Ai อัจฉริยะระดับโลก ดีที่สุดในโลก
โทนสีแอพเป็นโทนสีเข้ม theme สีทอง ดูมีราคาแพงระดับโลก ระดับเทพสุดๆ สวยหรู คุ้มค่าเงินที่ลูกค้าจ่าย ใช้แล้วดูเทห์ ออกแบบแอพให้ใช้ง่าย คนใช้แล้วติดวางไม่ลง และแนะนำคนอื่นให้มาใช้อีก

ออกแบบหน้าจอแอพให้มีขนาดเท่าจอมือถือ สเกลมาตรฐาน 390x844
icon เป็น 3D สวยงามดูแพง ปุ่มให้เป็น 3D สวยงามดูแพง ไล่เฉดสี 
ปรับปุ่มและ icon ให้พอดีหน้าจอ สมส่วน ดูสวยงามลงตัว เหมือนรูปภาพที่แนบ 100% เก็บรายละเอียดทุก Pixel

ออกแบบ Ui ให้มี 5 หน้า คือ Home / Aura Scan / Chat AI / Horoscope / Profile
เป็นแถบ icon 3D ด้านล่าง 5 tab และตัวหนังสือตามที่พิมพ์
เหมือนรูปตามไฟล์แนบ 100% เหมือนอย่างละเอียดที่สุดทุก pixel ตามไฟล์แนบ 100%

รายละเอียดหน้า Ui

หน้าที่ 1 โฮม
เป็นหน้าจอหลักที่แสดง 5 ฟีเจอร์ในหน้านี้ จัดขนาดทั้ง 5 ฟีเจอร์ให้เท่าๆกัน จัดตำแหน่ง ให้พอดีในหน้าจอมือถือ
เรียงตามลำดับจากบนลงล่าง ตามฟีเจอร์ต่อไปนี้
1 icon เป็นรูปกล้องถ่ายรูปสีทอง 3D  มีตัวหนังสือ "Aura Scan" สีทอง 3D ด้านข้างซ้ายมือ
2 icon หน้าผู้ชายสีทอง 3D  มีตัวหนังสือ "Chat AI" สีทอง 3D ด้านข้างซ้ายมือ
3 icon รูปดาวสีทอง 3D มีตัวหนังสือ "Horoscope" สีทอง 3D ด้านซ้ายมือ
4 icon รูป นั่งสมาธิสีทอง 3D ไม่เอารูปพระ มีตัวหนังสือ "Meditation" สีทอง 3D ด้านซ้ายมือ
5 icon ลูกแก้วกลมสีทอง 3D มีตัวหนังสือ "3D Meditate" สีทอง 3D ด้านซ้ายมือ

หน้าที่ 2 Aura Scan
ใช้กล่องโทรศัพท์ Scan หน้าคนแบบ realtime เพื่อเช็คพลังงานขณะนั้นอย่างแม่นยำที่สุด แล้วแสดงระดับพลังงาน จาก 0-10 (พลังงานต่ำ - พลังงานสูง) แสดงพลังงานเป็นแท่งบาร์ไล่ระดับสี พร้อมเก็บประวัติการเช็คพลังานลง วันที่ และ เวลา และ แถบพลังงาน เพื่อดูการพัฒนา

หน้าที่ 3 แชต Ai
เป็นภาพ Ai ภาพอยู่ตรงกลางจอ ในหน้านี้สามารถแชตเป็นข้อความได้ โดยมีภาพ icon ตามรูปที่แนบเหมือนคุยกับคนจริงๆ และยังสามารถพูดคุยด้วยเสียงมีไมค์ เอาไมค์ไว้ที่แถวเดียวกับข้อความแชต มี Ai ตอบเหมือนคุยกับคนจริงๆ โดยเอาคลังสมองจาก Hugging Face ที่เชื่อมต่อผ่าน Backend Proxy มาตอบให้ดีที่สุด

หน้าที่ 4 ดูดวง
โดยเอาข้อมูลจากโปรไฟล์ วันเดือนปีเกิด เวลาเกิด มาทำนายดวงด้วยโหราศาสตร์ที่ดีที่สุดในโลกที่แม่นยำที่สุดในโลก 4 ศาสตร์ สรุปออกมาเป็นคำทำนายเดียว ไม่ต้องแยกศาสตร์ ไม่ต้องมีให้เลือก แสดงข้อมูลการทำนายออกมาอย่างละเอียด แยกออกมาเป็นหมวดหมู่ ไม่เอา icon yin_yang ให้เอา icon อื่น โดยเอาข้อมูลคำนายจากคลังสมอง Hugging Face ที่เชื่อมต่อผ่าน Backend Proxy มาตอบให้ดีที่สุด

หน้าที่ 5 โปรไฟล์
เป็นหน้าที่แก้ไข ปรับแต่ง เก็บข้อมูลของผู้ใช้ทั้งหมด มีดังนี้
1 ชื่อ
2 รูปภาพ
3 email
4 วันเดือนปีเกิด เวลาเกิด
5 setup ภาษาได้ 50 ภาษาทั่วโลก
6 การเปลี่ยนแปลง password
7 การชำระเงินค่าแอพ และการต่ออายุรายปี รองรับการชำระเงินด้วยบัตรเครดิตทั่วโลก
8 การ upgrade สมาชิก ว่าใช้แบบประหยัด หรือแบบ Pro

จาก prompt ทั้งหมด ทำตามทุกตัวอักษร และให้เหมือนรูปตามไฟล์แนบ 100% 
เหมือนอย่างละเอียดที่สุดทุก pixel ตามไฟล์แนบ 100% 

*สำคัญ* เขียน code ให้แอพ mindverse เชื่อมต่อกับ Hugging face ผ่าน Backend Proxy ให้เรียบร้อย ไม่เอา HF_TOKEN ใส่ในแอพ ให้ใช้งาน  tab chat ai และ tab ดูดวงได้ อย่างเบ็ดเสร็จสมบูรณ์ออกเป็น Code อย่างสมบูรณ์ ให้พร้อม run ได้ทันที
